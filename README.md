## 📌 Tentang Proyek

Proyek ini merupakan aplikasi web interaktif yang menyajikan hasil **analisis kualitas perangkat lunak** menggunakan standar **ISO 9126**, dengan studi kasus perbandingan antara dua aplikasi populer:

- 🟢 **Duolingo** — aplikasi pembelajaran bahasa
- 🔵 **Tokopedia** — platform e-commerce

Analisis mencakup lima faktor kualitas utama ISO 9126 yang divisualisasikan secara interaktif, dilengkapi dengan skenario pengujian dan hasil uji black box.

---

## ✨ Fitur

- 📊 **Perbandingan Kualitas ISO 9126** — radar chart interaktif membandingkan kedua aplikasi pada 5 dimensi kualitas
- 🧪 **Uji Skenario** — dokumentasi skenario pengujian untuk setiap fitur
- 📋 **Uji Black Box** — hasil pengujian fungsionalitas berbasis input/output
- 💡 Tampilan responsif dan mudah dibaca

---

## 📐 Dimensi Kualitas ISO 9126

| Dimensi               | Deskripsi                                              |
|-----------------------|--------------------------------------------------------|
| **Fungsionalitas**    | Kemampuan aplikasi memenuhi kebutuhan pengguna         |
| **Keandalan**         | Konsistensi performa dalam berbagai kondisi            |
| **Kemudahan Penggunaan** | Seberapa mudah aplikasi dipahami dan digunakan      |
| **Efisiensi**         | Penggunaan sumber daya relatif terhadap performa       |
| **Portabilitas**      | Kemampuan berjalan di berbagai platform/lingkungan     |
